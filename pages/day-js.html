  • day.jsの使い方

Day.jsは軽量（2KB）で多くのブラウザでサポートされているJavaScriptの日付操作ライブラリです。

day.jsの使い方

下記コードでhtml内に
2020-09-06 Sun
2020-09-09 Wed
2020-09-16 Wed
と表記されます。 

//2020-09-06 Sun

//2020-09-09 Wed

//2020-09-16 Wed

日曜の午前かどうか判別するif文

let today = new Date();
if(today.getDay() == 0 && today.get.getHours() < 12 ){
	console.log('日曜の午前');
}
//　0 sun, 1 mon, 2 tue, 3 wed, 4 thu, 5 fri, 6 sat
Intl.DateTimeFormat 標準の日時書式指定文字列 他の国のローカルタイム（現地時刻）を文字列形式で取得する Day.jsで簡単に日付処理！
