命名ルール

クラス名からクラスの目的やスコープがわかるようにしておき、正確なコーディングに役立てる。

目次
  1. 命名ルール
  2. 略語

命名ルール

  • ローワーキャメル
  • ly- レイアウトグループ
  • bl- ブロックモジュール
  • el- エレメントモジュール
  • --　モディファイヤーの区切り
  • _ block element を繋ぐ

略語

  • container　単体のブロックを囲う時に使う　主にレイアウト的な入れ物　レイアウトに関するスタイルをつける
  • wrapper　単体のブロックを囲う時に使う　意味的な入れ物　レイアウト以外に関するスタイルをつける
  • _inner 複数のブロックを囲う時に使う　の下で使う入れ物　
  • btn ーbutton
  • txt ーtext
  • cnt ーcontent
  • ico ーicon
  • cat ーcategory
